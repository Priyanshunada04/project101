## Deployment
This site is hosted on Netlify.
- `index.html` — main website (single-file, all images embedded)
- `netlify.toml` — Netlify configuration & headers
- `_redirects` — www to non-www redirect
- `robots.txt` — SEO crawl instructions
- `sitemap.xml` — SEO sitemap
